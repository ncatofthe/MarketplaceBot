## Описание графического интерфейса

После запуска откроется окно с несколькими вкладками:

### Вкладка "Настройки"
Здесь вы настраиваете параметры работы ботов:

**Ozon:**
- "Включить бота Ozon" - активировать бота для Ozon
- API Key - ваш ключ API Ozon
- Company ID - ID вашей компании в Ozon

**Wildberries:**
- "Включить бота WB" - активировать бота для Wildberries  
- API Key - ваш ключ API Wildberries

**Общие настройки:**
- Интервал проверки (минуты) - как часто проверять новые отзывы (по умолчанию 60 мин)
- Мин. звезд для ответа - минимальная оценка, на которую бот будет отвечать (1-5, по умолчанию 1)
- Макс. ответов за один цикл - сколько ответов отправить за одну проверку (-1 = без ограничений)
- Задержка между запросами - пауза между отправкой ответов (секунды)

**Кнопки:**
- "Сохранить настройки" - сохранить введённые параметры
- "Запустить ботов" - начать работу
- "Остановить ботов" - остановить работу

### Вкладка "Шаблоны ответов"
Здесь вы можете редактировать тексты ответов:
- Загрузить шаблоны из файла
- Сохранить шаблоны в файл
- Сбросить на шаблоны по умолчанию

### Вкладка "Логи"
Показывает в реальном времени:
- Какие действия выполняет бот
- Ошибки и предупреждения
- Статус работы

Кнопки:
- "Очистить логи"
- "Сохранить лог в файл"

### Вкладка "Статус"
Показывает текущее состояние:
- Работает ли бот Ozon
- Работает ли бот Wildberries
- Кнопка обновления статуса

## Получение API ключей

### Ozon:
1. Зайдите в личный кабинет seller.ozon.ru
2. Перейдите в "Настройки" -> "API ключи"
3. Создайте новый ключ
4. Скопируйте "API Key" и "Client ID"

### Wildberries:
1. Зайдите в личный кабинет поставщика
2. Перейдите в "Настройки" -> "Доступ к API"
3. Создайте токен
4. Скопируйте токен

## Как использовать

### Шаг 1: Запуск
Запустите файл `dist\MarketplaceBot.exe` или `start_bot.bat` или выполните `python main.py`

### Шаг 2: Настройка
1. Перейдите на вкладку "Настройки"
2. Введите ваш API ключ для Ozon (если нужен)
3. Введите ваш API ключ для Wildberries (если нужен)
4. Установите интервал проверки (например, 60 минут)
5. Установите минимальное количество звезд (по умолчанию 1)
6. Нажмите "Сохранить настройки"

### Шаг 3: Запуск ботов
1. Отметьте галочками какие боты включить
2. Нажмите "Запустить ботов"
3. Перейдите на вкладку "Логи" для просмотра работы

### Шаг 4: Мониторинг
- Следите за логами во вкладке "Логи"
- Проверяйте статус во вкладке "Статус"
- Останавливайте ботов кнопкой "Остановить ботов"

## Сборка exe-файла

Если вы хотите пересобрать exe-файл:

1. Установите PyInstaller: `pip install pyinstaller`
2. Запустите сборку: `pyinstaller MarketplaceBot.spec`
3. Готовый файл будет в папке `dist\`

## Устранение проблем

### Ошибка "API ключ не найден"
- Проверьте правильность ввода API ключа
- Убедитесь что нажали "Сохранить настройки"

### Бот не отвечает на отзывы
- Проверьте логи на наличие ошибок
- Убедитесь что API ключи действительны
- Проверьте минимальное количество звезд

### Ошибка подключения к API
- Проверьте интернет-соединение
- Проверьте правильность API ключей
- Попробуйте перезапустить программу

### При запуске exe ничего не происходит
- Проверьте что в папке есть файл MarketplaceBot.exe
- Попробуйте запустить от имени администратора
- Проверьте наличие необходимых библиотек

Полный список возможных ошибок и их решений смотрите в файле "ИНСТРУКЦИЯ_ПО_ОШИБКАМ.md"
